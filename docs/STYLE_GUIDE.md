## 🏗️ Tipografía y Fuentes

### 1. Fuente Principal: **Aeonik**
Se utiliza para el cuerpo de texto y jerarquías generales.
* **Variantes disponibles:** Air, Light, Regular, Medium, Bold, Black.

### 2. Fuente de Marca: **Grotta Rounded**
* **Regla de Oro:** Usar **únicamente** para las palabras "Ready" o "Readymind".

### 3. Fuente de Acento: **Strokey**
* **Uso:** Para elementos decorativos o llamadas a la acción (CTAs).

---

## 🛠️ Notas de Implementación
* **Fondo recomendado:** Graphite (`#2d2d2d`) o Black (`#000000`).
* **Colores de resalte (Highlight):** Spring Green o Azure Blue.
